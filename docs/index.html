Features

Everything you need to
automate your digital life

From a single inbox action to complex multi-step workflows — OctaMind handles it all through natural language.

🧠
LLM-Powered Orchestration
Every natural language command is processed by GPT-4o mini, which intelligently selects and chains the right API actions — no rigid syntax needed.
🐙
Multi-Agent Platform
Create unlimited specialized agents from a central hub. Each agent runs in its own Streamlit window on a unique port, fully isolated.
💾
Persistent Agent Memory
6-layer memory system: personality, habits, working, episodic, semantic, and consciousness — each agent remembers its context across sessions.
⚡
10+ Gmail Automations
Set-and-forget automations: auto-delete spam, archive newsletters, daily email digest, VIP contact labeling, out-of-office replies, and more.
🎭
Personality Tuning
Customize each agent's tone, verbosity, humor, empathy, and proactiveness with precision sliders. Your agents, your style.
🔒
Safe by Design
Configurable max_operations guard prevents runaway bulk actions. OAuth-only access — your credentials never leave your machine.

How It Works

From command to action
in seconds

Three steps from your words to a completed task.

01
You type naturally
Type any request in plain English — "delete all emails from LinkedIn" or "send a reply to my boss". No commands to memorize.
02
LLM understands intent
GPT-4o mini analyses your request, selects the correct tool, and determines the exact parameters — max_operations guard keeps it safe.
03
Agent executes & remembers
The Gmail API action runs. Results are formatted and displayed. The interaction is stored in the agent's episodic memory for future context.

Quick Start

Up and running
in 3 minutes

Python 3.10+ required. Gmail OAuth credentials needed for the Gmail agent.

1
Clone & install dependencies
# Clone the repository
git clone https://github.com/you/octamind
cd octamind

# Create virtual environment
python -m venv .venv
source .venv/bin/activate

# Install all dependencies
pip install -r requirements.txt
2
Configure credentials
# Copy your Gmail OAuth file
cp ~/Downloads/credentials.json .

# Set GitHub Models API token (for LLM)
echo "GITHUB_TOKEN=ghp_..." > .env

# Verify setup
python -m pytest tests/ -q
153 passed in 30s ✓
3
Launch the Agent Hub
# Start everything
python start.py

✓ Dashboard → http://localhost:8501
✓ 3 agents loaded

# Or launch just the hub
python run_agent_hub.py
4
Create your first agent
# Or manage agents via Python
from src.agent.core import get_agent_manager

manager = get_agent_manager()
agent = manager.create_agent(
name="Email Butler",
agent_type="gmail",
role="Keep my inbox clean"
)
