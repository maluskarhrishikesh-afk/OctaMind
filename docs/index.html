Features

Everything you need to
automate your digital life

From a single inbox action to complex multi-step workflows — Octa Bot handles it all through natural language.

🧠
LLM-Powered Orchestration
Every natural language command is processed by GPT-4o mini, which intelligently selects and chains the right API actions — no rigid syntax needed.
🐙
Multi-Agent Platform
Create unlimited specialized agents from a central hub. Each agent runs in its own Streamlit window on a unique port, fully isolated.
💾
Persistent Agent Memory
6-layer memory system: personality, habits, working, episodic, semantic, and consciousness — each agent remembers its context across sessions.
⚡
Automations Per Agent
Set-and-forget automations built into every agent: auto-delete spam, daily digests, scheduled messages, VIP alerts, and more — running silently in the background.
🎭
Personality Tuning
Customize each agent's tone, verbosity, humor, empathy, and proactiveness with precision sliders. Your agents, your style.
🔒
Safe by Design
Configurable max_operations guard prevents runaway bulk actions. Tokens live in config/ on your machine — the LLM never sees them.
Why Octa Bot

Built different,
by design

Most AI assistant products run in the cloud, charge a monthly fee, and route your data through third-party services. Octa Bot takes the opposite approach.

🔗
No Platform Dependency
Octa Bot runs entirely on your own machine. You don't need to connect through any third-party automation platform, pay for integration middleware, or rely on an external service staying online. Clone, configure, run — that's it. Forever.
🛡️
Your Credentials, Your Machine
OAuth tokens, API keys, and access credentials stay in config/ on your computer — never uploaded, never shared. The LLM only ever receives task instructions. It cannot see, request, or infer your credentials. Not even once.
🚀
Near-Zero-Setup Personal AI
Clone the repo, drop your service tokens into config/ (the same tokens you already have), run python start.py. In minutes you have a personal AI that manages your email, files, WhatsApp, and Drive — fully local, no account to create, no usage limits.

How It Works

From command to action
in seconds

Three steps from your words to a completed task.

01
You type naturally
Type any request in plain English — "delete all emails from LinkedIn" or "send a reply to my boss". No commands to memorize.
02
LLM understands intent
GPT-4o mini analyses your request, selects the correct tool, and determines the exact parameters — max_operations guard keeps it safe.
03
Agent executes & remembers
The Gmail API action runs. Results are formatted and displayed. The interaction is stored in the agent's episodic memory for future context.

Quick Start

Up and running
in 3 minutes

Python 3.10+ required. Service credentials go into config/ on your machine — never anywhere else.

1
Clone & install dependencies
# Clone the repository
git clone https://github.com/you/Octa Bot
cd Octa Bot

# Create virtual environment
python -m venv .venv
source .venv/bin/activate

# Install all dependencies
pip install -r requirements.txt
2
Configure credentials
# Place your service credentials locally
cp ~/credentials.json config/credentials.json

# Set LLM provider token (task instructions only)
echo "GITHUB_TOKEN=ghp_..." > .env

# Confirm everything is wired correctly
python -m pytest tests/ -q
160 passed in 3s ✓
3
Launch the Agent Hub
# Start everything
python start.py

✓ Dashboard → http://localhost:8501
✓ 3 agents loaded

# Or launch just the hub
python run_agent_hub.py
4
Create your first agent
# Or manage agents via Python
from src.agent.core import get_agent_manager

manager = get_agent_manager()
agent = manager.create_agent(
name="Email Butler",
agent_type="gmail",
role="Keep my inbox clean"
)
