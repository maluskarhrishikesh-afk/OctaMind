How It Works

Three simple steps —
that’s all

No training, no manuals. If you can send a text message, you can use Octa Bot.

01
You say what you want
Type your request in plain language — “delete all LinkedIn emails” or “send this file to my boss”. No special commands. No technical words needed.
02
Octa understands & plans
Octa reads your message, figures out what needs to happen, and picks the right tools — safely and automatically. Big tasks are broken into steps.
03
Done — and remembered
The task is completed and you get a simple confirmation. Octa remembers the conversation, so it understands you better every time.

Get Started

Up and running
in minutes

You’ll need Python 3.10 or newer on your computer. Everything else is just a few commands.

1
Clone & install
# Get the code
git clone https://github.com/maluskarhrishikesh-afk/OctaMind
cd OctaMind

# Create a virtual environment
python -m venv .venv
source .venv/bin/activate

# Install everything needed
pip install -r requirements.txt
2
Add your credentials
# Drop your Google credentials here
cp ~/credentials.json config/credentials.json

# Add your AI token
echo "GITHUB_TOKEN=ghp_..." > .env

# Full guides are in /documentation/setup/
3
Launch Octa Bot
# Start everything
python start.py

✓ Dashboard → http://localhost:8501
✓ All helpers loaded and ready

# Open your browser and say hello!
4
Create your first helper
# Or create a helper via Python
from src.agent.core import get_agent_manager

manager = get_agent_manager()
agent = manager.create_agent(
name="My Email Helper",
agent_type="gmail",
role="Keep my inbox tidy"
)
